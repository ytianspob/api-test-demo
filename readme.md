# API Test Demo

基于 Postman + Newman + GitHub Actions 的 API 自动化测试项目。

---

## 项目简介

本项目用于演示 RESTful API 自动化测试流程，使用 Postman 编写接口测试用例，通过 Newman 在命令行执行，并结合 GitHub Actions 实现 CI/CD 自动化测试。

项目主要用于学习和实践：

- API 接口测试
- 自动化测试
- Newman 命令行执行
- GitHub Actions CI/CD
- HTML 测试报告生成

---

## 技术栈

- Postman
- Newman
- Node.js
- GitHub Actions
- JSONPlaceholder API
